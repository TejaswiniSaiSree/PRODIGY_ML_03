# PRODIGY_ML_03

## Cat and Dog Image Classification using SVM

This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs.

### Dataset
Microsoft PetImages Dataset:
https://www.microsoft.com/en-us/download/details.aspx?id=54765

### Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-Learn

### Features Implemented
- Image preprocessing
- Image resizing
- Feature extraction
- SVM image classification

### Output
- Classification accuracy for cat and dog images
